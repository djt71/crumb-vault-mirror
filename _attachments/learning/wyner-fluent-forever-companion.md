# Fluent Forever — Gabriel Wyner

**Purpose:** Language learning methodology reference — source material for the learning-plan skill's language skill type.

![[wyner-fluent-forever.pdf]]

## Notes
- Published 2014 by Harmony Books (Crown/Penguin Random House)
- ISBN 978-0-385-34811-9
- Author is an opera singer who learned 6+ languages using these techniques
- Core methodology: pronunciation → vocabulary (no translation) → grammar (via sentences) → immersion
- Heavy emphasis on Anki/SRS as the backbone tool
- Complements Make It Stick (empirical backbone) and Peak (deliberate practice framework)
