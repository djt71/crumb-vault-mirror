Quick cheat sheet — all commands start with Ctrl+A:

|   |   |
|---|---|
|**Action**|**Keys**|
|New tab|Ctrl+A c|
|Next tab|Ctrl+A n|
|Previous tab|Ctrl+A p|
|Go to tab 1-9|Ctrl+A 1, Ctrl+A 2, etc|
|Vertical split|`Ctrl+A|
|Horizontal split|Ctrl+A -|
|Detach (leave running)|Ctrl+A d|
|Reattach after SSH reconnect|tmux attach|
